שיעור 4: CSS Grid — בניית רשת חכמה

1) Grid — פריסה דו־ממדית (שורה ועמודה)

עד היום השתמשנו ב-Flexbox כדי לסדר דברים בשורה או בעמודה. בשיעור זה נכיר את ה-Grid, שמאפשר לנו לשלוט בשניהם בו-זמנית. הוא מושלם למבנים כמו גלריות, חנויות דיגיטליות או לוחות משחקים.

איך נראה הקוד שלנו? (דוגמה: חנות נעליים)

כדי שהגריד יעבוד, אנחנו מגדירים פקודות למכולה (החנות) ופקודות לפריטים (הנעליים).

דוגמה לקוד CSS:

/* 1. המכולה (החנות כולה) */ .shoe-store { display: grid; /* יצירת 3 עמודות שוות */ grid-template-columns: repeat(3, 1fr); gap: 20px; background-color: #f0f0f0; padding: 15px; } /* 2. הפריט הבודד (נעל אחת) */ .shoe-item { background-color: white; height: 200px; border: 1px solid gray; }

2) מי המכולה ומי הפריט?

כדי שה-CSS יעבוד, אנחנו חייבים לקשר אותו ל-HTML בעזרת ה-class.

ב-HTML זה ייראה כך:

<div class="shoe-store"> <div class="shoe-item">נעל א'</div> <div class="shoe-item">נעל ב'</div> <div class="shoe-item">נעל ג'</div> </div>

ה-class שנקרא shoe-store הוא המכולה: הוא קובע שיש 3 עמודות בחנות.

ה-class שנקרא shoe-item הוא הפריט הבודד: הוא קובע איך תיראה כל קופסת נעל (צבע לבן, גובה 200 פיקסלים).

3) שילוב תאים (Spanning) — פריט "ענק"

לפעמים נרצה שפריט אחד (למשל שלט מבצעים) יהיה רחב יותר מהשאר.

שימו לב: את הפקודה הזו כותבים ב-class של הפריט עצמו!

מאפיין מה זה עושה? דוגמה
grid-column כמה עמודות הפריט יתפוס span 3 (יתפוס שורה שלמה של 3 עמודות)

דוגמה לשלט מבצעים שתופס את כל רוחב החנות:

.sale-banner { grid-column: span 3; /* נמתח על פני 3 עמודות */ background-color: red; color: white; text-align: center; }

4) תרגול — בונים לוח משחק זיכרון (מאפס!)

עכשיו תורכם ליישם את מה שלמדנו על עולם תוכן חדש: משחק זיכרון. אל תעתיקו מהדוגמה למעלה, אלא תנסו לחשוב מה המכולה שלכם ומה הפריטים.

תרגיל 4.1 — יצירת לוח המשחק (המכולה)

  • צרו div ב-HTML עם class בשם memory-game.
  • הכניסו פנימה 16 אלמנטים מסוג div (אלו יהיו הקלפים).
  • ב-CSS: הגדירו למכולה (memory-game) גריד של 4 עמודות (רמז: השתמשו ב-repeat).
  • הוסיפו gap של 12px.

תרגיל 4.2 — הוספת כותרת ללוח (Spanning)

  • הוסיפו div ראשון בתוך המכולה (לפני הקלפים) עם הטקסט: "הצלחתי לבנות גריד!".
  • תנו לו class בשם main-title.
  • ב-CSS: גרמו לכותרת להימתח על פני כל 4 העמודות (רמז: grid-column).
  • תנו לכותרת צבע רקע צהוב כדי שנראה שהיא באמת תופסת את כל השורה.

תרגיל 4.3 — עיצוב קלפי המשחק (הפריטים)

  • צרו ב-CSS הגדרה ל-class בשם memory-card.
  • תנו לקלף גובה (height) של 130px.
  • תנו לו צבע רקע כחול כהה ועיגול פינות (border-radius: 10px).
  • חזרו ל-HTML וודאו שלכל 16 ה-div-ים של הקלפים יש את ה-class="memory-card".

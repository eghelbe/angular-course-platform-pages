שיעור 2 — CSS: Properties בסיסיים ותרגול

עובדים ב־JSFiddle:
https://jsfiddle.net

משתמשים באותו HTML מהשיעור הקודם — לא משנים את מבנה ה־HTML, רק מוסיפים CSS.

1) מה זה CSS?

CSS היא שפה שאחראית על העיצוב של דף האינטרנט.

היא קובעת:

  • צבעים
  • ריווחים
  • גדלים
  • מסגרות
  • מראה כללי

CSS לא משנה את המבנה של הדף.
המבנה נקבע ב־HTML, והעיצוב נקבע ב־CSS.

2) תחביר (Syntax) של CSS

selector {   property: value; }

selector — על איזה אלמנט או מחלקה העיצוב חל
property — איזה מאפיין אנחנו משנים
value — הערך של אותו מאפיין

כל שורה מסתיימת בנקודה־פסיק (;).

3) דוגמה מלאה ל־CSS תקין

.dark-title { background-color: black; color: white; }

4) חיבור בין CSS ל־HTML (class)

כדי לחבר מחלקה מה־CSS לאלמנט ב־HTML, משתמשים במאפיין class.

<h2 class="dark-title"> כותרת עם רקע שחור וכתב לבן </h2>

5) חיבור של מחלקה אחת לכמה אלמנטים

מחלקה ב־CSS לא מיועדת לאלמנט אחד בלבד.
אפשר לחבר את אותה מחלקה לכמה אלמנטים שונים בדף.

זה חשוב, כי כך אפשר:

  • לעצב הרבה אלמנטים
  • בעזרת הגדרה אחת בלבד ב־CSS
.notice {   color: blue;   font-size: 18px; } <p class="notice">זהו טקסט חשוב</p> <h3 class="notice">כותרת חשובה</h3> <div class="notice">הודעה כללית</div>

אם משנים את המחלקה notice — כל האלמנטים שמחוברים אליה משתנים יחד.

6) חיבור של כמה מחלקות לאותו אלמנט

אפשר להחיל יותר ממחלקה אחת על אותו אלמנט.

שמות המחלקות:

  • נכתבים עם רווח ביניהם
  • ועדיין בתוך המרכאות של class
<h2 class="dark-title italic"> כותרת עם רקע שחור וכתב לבן בסטייל נטוי </h2>

8) מבנה הקופסה (Box Model)

ב־CSS, כל אלמנט בדף הוא קופסה. גם כותרת, גם פסקה, גם תמונה וגם <div>.

מבנה הקופסה מורכב מארבע שכבות, מהפנימית לחיצונית:

  • Content
  • Padding
  • Border
  • Margin

עברו עם העכבר מעל השכבות וראו איזו שכבה מודגשת.

Margin
Border
Padding
Content

אם מבינים את מבנה הקופסה — מבינים איך אלמנטים יושבים בדף ואיך לשלוט בעיצוב שלהם.

9) תרגול

תרגיל 1 — עיצוב הדף מהשיעור הקודם

⚠️ חשוב מאוד
אחרי כל שינוי שאתם עושים:
  • הסתכלו על הדף
  • נסו להבין מה בדיוק השתנה
המטרה היא: לראות --> להבין --> לצבור ביטחון

עבדו עם הקוד הבא (ב־JSFiddle):

<main>   <h1>כותרת ראשית</h1>   <section>     <h2>אזור ראשון</h2>     <h3>כותרת משנה לאיזור הראשון</h3>     <p>טקסט לאיזור הראשון</p>     <img src="https://via.placeholder.com/300">   </section>   <section>     <h2>אזור שני</h2>     <h3>כותרת משנה לאיזור השני</h3>     <p>טקסט לאזור שני</p>   </section>   <section>     <h2>אזור שלישי</h2>     <h3>כותרת משנה לאיזור השלשי</h3>     <p>טקסט לאזור שלשי</p>   </section> </main>

בצעו את המשימות:

  • צבע שונה ל־h1, h2, h3
  • background-color לכל section
  • padding לכל section
  • margin בין ה־sections
  • מסגרת בעזרת border-width, border-style, border-color
  • border-radius
  • מחלקה לתמונה: width: 250px, height: 250px

תרגיל 2 — קופסאות

  • צרו 8 אלמנטים <div>
  • הגדירו מחלקה box עם:
    • width: 100px
    • height: 100px
    • border-width: 2px
    • border-style: solid
    • border-color: gray
  • הגדירו מחלקות צבע
  • חברו לכל קופסה: box + מחלקת צבע אחת
  • הוסיפו margin, padding, border-radius

שיעור 3 – CSS Layout

1) display

לכל אלמנט HTML יש ערך display ברירת מחדל. גם אם לא כתבנו עליו CSS בכלל.

display: block

ברירת המחדל של האלמנטים:
<div>, <p>, <section>
היא display: block - הם תופסים שורה שלמה. 

<div class="item">1</div>
<div class="item">2</div>

.item {
  display: block;
}
1
2

display: inline

ברירת המחדל של <span>. זורם בתוך הטקסט ולא מקבל רוחב או גובה. 

<span class="item">1</span>
<span class="item">2</span>

.item {
  display: inline;
  width: 100px; /* יתעלם מהרוחב! */
}
1 2

display: inline-block

יושב בשורה כמו inline, אבל כן מקבל width / height. 

<div class="item">1</div>
<div class="item">2</div>

.item {
  display: inline-block;
  width: 80px;
  height: 80px;
}

2) CSS Flex Container

Flexbox עובד על מכולה וילדים. מגדירים Flex רק על המכולה — והיא שולטת על סידור הילדים. 

/* HTML */
<div class="container">
  <div class="item">1</div>
  <div class="item">2</div>
  <div class="item">3</div>
</div>

/* CSS */
.container {
  display: flex;
}

מאפייני השליטה של Flex Container

  • flex-direction — קובע את כיוון הסידור הראשי של הילדים (שורה / עמודה).
  • justify-content — שולט על פיזור הילדים לאורך הציר הראשי.
  • align-items — שולט על יישור הילדים על הציר המשני.
  • flex-wrap — קובע האם ילדים נשברים לשורה חדשה או נשארים בשורה אחת.
  • flex-flow — קיצור דרך לשילוב של flex-direction ו־flex-wrap.

עכשיו, אחרי שאנחנו מכירים את כלי השליטה — נעבור עליהם אחד אחד, עם דוגמאות.

10) תרגול מסכם — Flexbox

בתרגילים הבאים תתרגלו שליטה במכולת Flex באמצעות מה שלמדנו:

  • flex-direction
  • justify-content
  • align-items
  • flex-wrap
  • flex-flow

⚠️ חשוב
בכל התרגילים: מגדירים display: flex רק על המכולה

תרגיל 1 — סרגל ניווט אופקי (Header)

צרו סרגל ניווט אופקי, כמו בראש אתר.

דרישות:

  • צרו מכולה אחת
  • בתוך המכולה:
    • לוגו בצד אחד
    • 5 כפתורים / קישורים בצד השני
  • הסרגל צריך להיות:
    • אופקי
    • עם רווח בין הלוגו לכפתורים
    • מיושר יפה על הציר המשני

? חשבו איזה מאפיין שולט על הפיזור בציר הראשי,
ואיזה מאפיין שולט על היישור בציר המשני.
חשבו האם צריך יותר ממכולה אחת.

תרגיל 2 — גלריה של ריבועים

צרו גלריה גמישה של ריבועים צבעוניים.

דרישות:

  • מכולה אחת
  • 20 ריבועים (אלמנטים)
  • לכל ריבוע:
    • רוחב קבוע
    • גובה קבוע
    • צבע רקע שונה
  • הגלריה צריכה:
    • להישבר לשורות
    • להיות ממורכזת על הציר הראשי
    • ולהיות ממורכזת גם על הציר המשני

? בדקו מה ההבדל בין justify-content לבין align-items.

תרגיל 3 — סרגל ניווט תחתון (Footer)

צרו סרגל ניווט אופקי נוסף, בסגנון שונה.

דרישות:

  • סרגל אופקי
  • 3 קישורים בלבד:
    • תנאי פרטיות
    • מדיניות השימוש
    • צור קשר
  • הסרגל צריך:
    • להיות ממורכז על הציר המשני
    • לפזר את הקישורים ברווחים שווים על הציר הראשי

? חפשו ערך ב־justify-content שמפזר רווחים בצורה שווה.

תרגיל 4 — מבנה אתר מלא (תרגיל מסכם)

בתרגיל זה תחברו את כל התרגילים הקודמים למבנה אתר אמיתי.

מבנה האתר:

  • בחלק העליון — סרגל הניווט מתרגיל 1
  • מתחתיו — הגלריה מתרגיל 2
  • בתחתית — סרגל הניווט מתרגיל 3

דרישות חשובות:

  • כל האתר צריך להיות מסודר בצורה אנכית
  • הגלריה צריכה לגדול ולמלא את רוב גובה הדף
  • סרגל ה־Footer צריך להישאר בתחתית הדף
  • אין להשתמש ב־position
  • הכול מתבצע בעזרת Flexbox בלבד

? חשבו מי המכולה הראשית של האתר, ואיך היא שולטת על שלושת החלקים שבתוכה.
